# CipherSolver

CipherSolver is a personal project developed to deepen my understanding of Python and its standard library. The project was inspired by ARGs (Alternate Reality Games) and explores how data can be transformed using classical ciphers, encoding techniques, and cryptographic hash functions.

The entire application is terminal-based, focusing on simplicity, efficiency, and ease of use.

## Features
### Ciphers
- Atbash
- Caesar
- Keyboard
- Morse
- Vigenère
### Encodings
- ASCII
- Base64
- Binary
- Hexadecimal
-  ROT13
- URL Encoding
### Hash Functions
- Blake2b
- MD5
- SHA-256
- SHA-512
## Installation

CipherSolver requires no additional dependencies.

git clone https://github.com/GuilhermeYuutaMimura/CipherSolver.git
cd CipherSolver
python main.py
### Usage

CipherSolver allows users to encrypt and decrypt text using the available ciphers and encoding methods, as well as generate with the availables hashing algorithms.

The entire application is designed to be operated exclusively with the keyboard, primarily using the number keys and the Enter key, providing a fast and straightforward terminal experience.
